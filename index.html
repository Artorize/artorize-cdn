Artorize - AI-Protected Image Viewer

How it works

This page displays a protected image with its mask overlay rendered using the SAC v1 protocol.

  • The polluted image is loaded from the CDN
  • The mask data is fetched from [image-url].sac
  • The mask is parsed and rendered on a canvas overlay
  • The canvas is positioned absolutely over the image to show the protection mask

Note: Set the src attribute of #protected-image to your CDN image URL to test.
